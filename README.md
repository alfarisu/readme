## 🚀 Featured Projects

### 🎯 **Helpdesk System Web**
**Intelligent IT ticket management with ML-powered automation**

Production-grade helpdesk system leveraging machine learning for intelligent issue prioritization. The system analyzes incoming tickets and automatically assigns priority levels with 87% accuracy, significantly reducing manual triage time and improving response efficiency.

**Key Features:**
- ML-powered auto-priority detection using TF-IDF + Logistic Regression
- Real-time SLA tracking with breach alerts
- Role-based access control (Admin, Tech Lead, Support Staff)
- Comprehensive dashboard with analytics
- Automated workflow orchestration

**Tech Stack:**
```
Backend: Laravel 12 API | Frontend: Next.js 16 | ML: Python (scikit-learn)
Database: PostgreSQL | Architecture: Microservices pattern
```

**Impact:**
- 87% accuracy in automatic priority classification
- 40% reduction in ticket processing time
- Real-time SLA compliance monitoring

→ **[View Repository](https://github.com/risuunava/helpdesk-system-web)**
